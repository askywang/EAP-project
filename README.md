# 植物盖度分析工具

一个用于计算卫星图像中红色边界区域内植物盖度的Python脚本。

## 功能

- 自动检测红色边界线围成的校园区域
- 自动检测绿色植被区域
- 计算植物盖度、植物面积和校园总面积
- 支持手动输入比例尺
- 生成可视化分析结果图

## 安装依赖

```bash
pip install opencv-python pillow matplotlib numpy
```

## 使用方法

### 方法1：交互式输入（推荐）

```bash
python3 vegetation_coverage_final.py "图片名.png"
```

运行后会提示输入比例尺信息：
```
============================================================
请输入比例尺信息
============================================================
当前图片: 截屏2026-03-10 00.47.12.png

请查看图片左上角的比例尺，然后输入以下信息：
比例尺像素长度 (像素): 223
比例尺实际长度 (米): 100
```

### 方法2：命令行直接指定

```bash
python3 vegetation_coverage_final.py "图片名.png" 223 100
```

参数说明：
- 第1个参数：图片路径
- 第2个参数：比例尺像素长度
- 第3个参数：比例尺实际长度（米）

## 输入图片要求

- 图片格式：PNG、JPG等常见图片格式
- 图片中需要有红色边界线（纯红色 #FF0000）围成的区域
- 左上角需要有比例尺（红色线段）

## 如何测量比例尺

1. 打开图片，查看左上角的比例尺
2. 使用图片查看器或截图工具测量比例尺的像素长度
3. 记录比例尺标注的实际长度（通常为50米或100米）

## 输出结果

脚本会输出以下信息：

```
============================================================
ANALYSIS RESULTS
============================================================

🌿 Vegetation Coverage: 29.47%
🌱 Vegetation Area: 49235.46 m² (4.92 ha)
🏫 Campus Total Area: 167080.17 m² (16.71 ha)
============================================================
```

同时生成可视化结果图：
- 原始图像
- 植被掩码（绿色区域）
- 分析结果统计

结果图保存为：`原图片名_result.png`

## 技术原理

1. **红色边界检测**：使用HSV颜色空间检测纯红色（#FF0000）边界线
2. **校园区域提取**：通过形态学操作和轮廓检测提取红色边界围成的区域
3. **植被检测**：使用HSV颜色空间检测绿色植被区域
4. **面积计算**：根据比例尺将像素面积转换为实际面积

## 注意事项

- 确保图片中的红色边界线清晰可见
- 比例尺输入要准确，否则面积计算会有偏差
- 如果检测结果不理想，可以查看 `debug_scale_region.png` 和 `debug_scale_red.png` 进行调试
